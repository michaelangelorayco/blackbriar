You will be given a unique cadet id, and you will be required to append your cadet id in every resource that you will create for this lab activity.
For the instructions below, we use A0 as an example cadet id, your cadet id will be different, be very mindful of this.

Good luck!

1. Check if you are in the Singapore region, if not please do switch region.


2. Navigate to CloudWatch.


3. Click on create dashboard and name it dashboard followed by your cadet id separated by a dash.



Click on create dashboard.




4. Select any widget that you want.

For this example, I will select line, then click configure.





5. Select any metric that is available on your account.



For this example, I will select S3.





6. Try to adjust the settings to display a chart. For this example, I selected a bucket with the metric name NumberOfObjects and set to display 3days worth of data.



Click create widget.




7. You have created a dashboard! Please take note of the name of your dashboard name, we will ask for it before you end the lab.
